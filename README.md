## üì¶ Installation

1. **Clone the repository**
   ```bash
   git clone <repo-url>
   cd "MERN Admin Dashboard"
   ```

2. **Setup Backend**
   ```bash
   cd backend
   npm install
   # Create .env file with PORT, MONGO_URI, JWT_SECRET
   npm run dev
   ```

3. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Access the App**
   - Frontend: `http://localhost:5173`
   - Backend API: `http://localhost:5000`
